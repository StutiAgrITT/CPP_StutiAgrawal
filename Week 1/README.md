## Steps to Run butterfly_pattern.cpp

### 1. Open Terminal
Navigate to the folder where this file is present.
`cd path/to/folder`

### 2. Compile the program
`g++ butterfly_pattern.cpp -o butterfly_pattern`

### 3. Run the Program
`./butterfly_pattern`

### 4. Enter input
Enter the number of the columns in left triangle in the pattern
